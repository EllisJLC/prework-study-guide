## Installation

Create new branch:

git checkout -b -branch name-

1. Add files to commit: (A for all)
git add -A

2. Save snap of files:
git commit -m "-commit message-"

3. Sync:
git pull origin main
 
4. Upload:
git push origin -branch-

5. Merge branch with main in github.
